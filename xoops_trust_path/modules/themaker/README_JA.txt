Themaker
========
XOOPS 用のテーマの雛形を出力するモジュールです。


必要な環境
----------
- XOOPS Cube Legacy 2.2
- http://colorschemedesigner.com/ (カラースキーム)


主な機能
--------
- パラメタに従ってテーマを出力
  - HTML バージョン（xhtml, html5）
  - レイアウトタイプ（グリッド960px, Elastic）
  - カラム数（1-3）
  - センターブロックの位置（コンテンツ下、フッタ）
  - カラースキーム
